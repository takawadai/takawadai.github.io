〇アプリケーションの概要
和歌山大学の案内をしてくれるアプリケーションです
主に、各学部の情報についてのサイトを表示してくれます。

〇使用方法
startボタンを押した後に、
「システム工学部の就職は？」
「和歌山大学の学部は？」などとしゃべりかけてみてください。
stopボタンを押すと音声認識が終わります。

〇アピールポイント
・想定外の質問が来たら、自然言語WebAPIが応答するようにしました。

・正規表現を用いることで、「シス工」や「和大」など、APIの辞書に登録されていない和歌山大学オリジナルの固有名詞にも反応するように
しました


・受験生が使うことを想定しているため、勉強法についての質問にも対応しました。

・「システム工学部」という学部名は珍しく、知らない人が多いため、たとえば「工学部の就職先は？」と尋ねても応答できるようにしました。
また、「メジャー」も同様に知らない人が多いため、質問する人が「システム工学部の学科は？」と聞いても応答できるようにしました。
